# 🍅⏱️ fo’kus - Study Timer

Hello, world! 👋 O projeto ainda está em construção 🚧, mas já pode ser acessado <a href="https://fokus-study-timer.vercel.app" target="_blank">aqui.</a>

Este é um aplicativo Pomodoro Timer desenvolvido com o objetivo de estudo e prática de React utilizando Vite e Tailwind CSS.

O projeto ajuda na organização do tempo e no aumento da produtividade, permitindo alternar entre ciclos de foco, pausa curta e pausa longa, seguindo a técnica Pomodoro.
